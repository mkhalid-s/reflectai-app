## Archive Policy

Documents are archived when they:
- Are no longer relevant to current development
- Document resolved issues or deprecated features
- Are tactical/temporary guides that have served their purpose
- Are superseded by newer documentation

**Retention**: Archived docs are kept indefinitely for historical reference.

**Review**: Archived docs are reviewed quarterly for potential deletion.

---

## Need Something from the Archive?

If you need information from an archived document:
1. Check if the information exists in current documentation
2. If not, review the archived document
3. Consider updating current docs if the information is still valuable
4. Contact the documentation team if you have questions

---

**Note**: Do not rely on archived documents for current development. Always refer to the main documentation directory for up-to-date information.
