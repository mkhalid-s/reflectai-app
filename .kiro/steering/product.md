# ReflectAI Product Overview

ReflectAI is a cloud-native, multi-agent competency analysis platform that transforms traditional competency assessment through intelligent agent orchestration. The platform integrates with Slack to provide AI-powered career development insights and competency analysis.

## Core Functionality

- **Multi-Agent System**: A system of coordinated agents orchestrated by Temporal.io workflows. Phase 1 includes two core agents (Analysis and Advisor) with a clear path to more specialized roles.
- **Slack Integration**: Native Slack bot with both Socket Mode and HTTP mode support
- **Competency Analysis**: Intelligent analysis of user skills, career progression, and development recommendations
- **Report Generation**: Automated generation of competency reports and career development plans

## Key Features

- Event-driven architecture with intelligent routing between single and multi-agent processing
- Real-time Slack interactions with conversation context management
- Comprehensive observability with OpenTelemetry integration
- Enterprise-grade security with OAuth2/OIDC authentication
- Flexible deployment options (Docker, Kubernetes)

## Target Users

- HR professionals and managers seeking competency insights
- Employees looking for career development guidance
- Organizations implementing competency-based development programs
